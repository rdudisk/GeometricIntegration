Librairies tierces à installer:
*******************************
	
	TRILINOS

	# apt-get install trilinos-all-dev trilinos-doc (il y a peut-être moyen d'être plus spécifique que ça...)


	EIGEN 3

	# apt-get install libeigen3-dev libeigen3-doc
	or, for the up-to-date version
	$ cd ~
	$ git clone https://github.com/eigenteam/eigen-git-mirror
	$ cmake eigen-git-mirror
	# make install

	Dans tous les cas, il faudra probablement faire
	# ln -s /usr/include/eigen3/Eigen /usr/local/include/Eigen
	ou alors préciser à chaque fois
	$ g++ ... -I/usr/include/eigen3 ...

	Nécessaire ? Essayer sans d'abord
	# apt-get install libmrpt-dev


	DOXYGEN

	# apt-get install doxygen*
